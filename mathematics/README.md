* 进制： 土财主家的傻儿子上学，一、二、三，

    * 10进制 - 人类的手指头，
    * 12进制 (除大拇指外的指关节数量)，
    * 60进制 (左手5个手指头，右手除大拇指外的指关节数量)，
    * 2进制 (计算机领域)
    * 16进制 (半斤八两)
